About us

Lyre is an Open-source project that aims to provide a wide range of free multimedia programs.

The main goal for Lyre Project is to allow the end user to download content from the Internet for personal use and offline viewing. Lyre Project currenty provides a YouTube downloader and downloads history viewer. Next goal is to create a complete software suite which will include a downloader, file library, media player and more.

Vision

I have been using different music/video programs and downloaders for quite some time before I got this idea. There were many things that were bothering me. Music programs lacked the control I wanted. I didn't like the UI or some other part of the user experience. Downloaders were a slightly different story. The main thing that bothered me were ads, adware, forced installations of other programs, and overall shady behavior.

One day I have had enough and made a plan to create my own program that would be consistent and would fix the features that bothered me. I also decided that the project would be Open-source. There are two reasons behind that decision. The first reason is sharing the code and allowing others to improve upon my work. The second reason is really important in my opinion - transparency. Transparency improved the trust in the system and if a value that means a lot to me. I also think that it should be implemented elsewhere.
